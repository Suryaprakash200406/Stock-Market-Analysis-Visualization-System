# Stock Market Analysis and Visualization System

Stock Market Analysis and Visualization System is a web-based application that fetches stock market data from NSE India, cleans offline and online data, and displays the processed results for analysis and visualization. It supports both historical and real-time stock information.

## Technologies Used
- Java (NetBeans IDE)  
- JSP, Servlet, HTML5, JavaScript  
- SQL, MySQL  
- Apache Tomcat Server  

## Key Features
- Fetches real-time stock data from NSE India.  
- Cleans and normalizes offline CSV stock data files.  
- Displays processed stock information and trends in a web interface.  
- Supports both online and offline data updates.  

## How to Run
1. Open the project in **NetBeans IDE**.  
2. Ensure **Apache Tomcat** is configured as your server.  
3. Right-click the project folder and select **Run**.  
4. The application will launch in your default web browser.  

## Project Structure
- `src/` – Java source code files  
- `web/` – Web interface files  
- `nbproject/` – NetBeans project configuration  
- `build.xml` – Build configuration  
- `README.md` – Project documentation
